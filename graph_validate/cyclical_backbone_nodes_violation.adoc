

## Test: Backbone is acyclical

#### Test description

This test searches for directional cycles back to the same node.

#### The test
[source,cypher]
----
MATCH p = (a)-[:DERIVED_FROM *]->(a)
WHERE a.submissionID = '{submissionID}'
RETURN a
----
